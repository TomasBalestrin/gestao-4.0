Design System Gestão 4.0

Catálogo vivo dos tokens, componentes e padrões reais usados no sistema. Construído sobre Tailwind CSS + shadcn/ui (Radix UI), tipografia Source Serif 4 nos títulos e Inter no corpo, paleta inspirada no Claude (creme quente / near-black) com acento azul. Light e dark mode habilitados via classe .dark no <html>.

Fundações

Tokens semânticos, paletas de apoio, tipografia, raio, espaçamento e sombras.

01

Cores semânticas (CSS variables)

Todas as cores do produto vêm dessas variáveis HSL no app/globals.css. Light e dark trocam pela classe .dark no <html>.

Paleta pastel — etapas do kanban

10 cores tom claro usadas no header das colunas (tintBg(cor, 0x33) ≈ 20% alpha). Vão pra etapas.cor quando o admin cria uma nova.

Tipografia

Duas famílias: Source Serif 4 nos títulos (editorial, peso 500–600) e Inter no corpo. Mono é JetBrains Mono para IDs e timestamps.
